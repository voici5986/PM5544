# PM5544 测试卡参数编辑器

[English](./README.md) | [中文](./README.zh-CN.md)

一个单文件 HTML 工具，用于预览并导出可配置文字栏的 PM5544 风格测试卡。

---

## 项目简介

这是一个轻量、无需构建的 PM5544 风格图形编辑器。  
直接在浏览器打开 `index.html`，即可修改文字栏参数、实时预览，并导出：
- 自定义 SVG
- 自包含 HTML

---

## 这个项目解决什么问题

- 快速生成带台标/时间/日期叠加信息的 PM5544 风格画面。
- 通过 URL 参数分享可复现配置。
- 导出文件可移植（资源内嵌，不依赖外部字体或图片路径）。

---

## 主要功能

- 单文件应用（`index.html`），无框架、无安装步骤。
- 4:3 响应式预览画布。
- 画布低于最小阈值时自动显示 `no signal`。
- 文字栏参数控制：
- `tbar`（顶部单行）
- `tbar1` + `tbar2`（顶部双行）
- `bbar`（底部左下栏）
- `bbarfull`（底部中栏）
- 优先级规则：`tbar` 会覆盖 `tbar1/tbar2`。
- 动态关键字：
- `TIME` 显示实时钟表
- `DATE` 显示实时日期
- 导出能力：
- SVG（内嵌底板 + 内嵌字体）
- HTML（单文件自包含导出）

---

## 参数说明

| 参数 | 说明 |
|---|---|
| `tbar` | 顶部大字单行 |
| `tbar1` | 顶部小字双行上行 |
| `tbar2` | 顶部小字双行下行 |
| `bbar` | 底部左下栏 |
| `bbarfull` | 底部中栏 |

特殊值：
- `TIME` -> 实时时间
- `DATE` -> 实时日期

---

## 导出行为说明

- **导出的 HTML** 是自包含文件，`TIME/DATE` 会持续动态更新。
- **导出的 SVG** 内嵌资源，并在支持脚本的 SVG 查看环境中可动态更新 `TIME/DATE`。
- 若目标环境禁止 SVG 脚本（例如部分图片处理链路），动态文本会退化为静态显示。

---

## 本地使用

1. 用现代浏览器打开 `index.html`。
2. 填写任意参数。
3. 点击：
- `应用并更新 URL`：将当前配置写入 URL。
- `导出 SVG` 或 `导出 HTML`：导出结果文件。
